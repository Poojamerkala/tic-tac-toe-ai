# Tic Tac Toe AI

This project is part of the CodSoft Artificial Intelligence Internship.

## Description
This is a Tic Tac Toe game where a human player competes against an AI.  
The AI uses the Minimax algorithm to choose the best possible move.

## Features
- Human vs AI gameplay
- Minimax algorithm
- Command line interface
- Unbeatable AI

## Technologies Used
- Python

## How to Run

1. Clone the repository
2. Run the Python file

python tic_tac_toe.py
